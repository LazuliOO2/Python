# Repositório de Projetos em Python

Este repositório reúne projetos desenvolvidos exclusivamente em Python.  
Projetos que utilizam outras linguagens são armazenados em repositórios separados.

---

## Projetos

### Análise e Previsões do Naufrágio do Titanic
**Descrição:** Análise exploratória do dataset do Titanic, com previsões de sobrevivência utilizando modelos de machine learning.

---

### Web Scraping com Selenium
**Descrição:** Extração automatizada de dados de sites dinâmicos utilizando Selenium.

---

### Análise de Ações com Streamlit e API do Gemini
**Descrição:** Aplicação em Streamlit para análise de ações integrada à API do Gemini para insights adicionais do mercado financeiro.

---

### Aula de Encerramento
**Descrição:** Atividade acadêmica composta por duas partes:  
- **Análise de Livro (POO):** exploração de conceitos de Programação Orientada a Objetos aplicados a uma obra literária.  
- **Versão adaptada:** releitura personalizada refletindo cenários mais realistas.

---

### ETL – Mercado Livre
**Descrição:** Pipeline ETL que coleta dados do Mercado Livre, trata e salva em SQL e exibe dashboards interativos com Streamlit.

---

### portfolioAtivo
**Descrição:** Desenvolvimento de algoritmos de aprendizado por reforço para mercado financeiro (A2C, PPO e PPO + GEN).

---

### arquivoanalise
**Descrição:** Dashboard automatizado para análise de usuários e identificação de padrões de compra acima da média.

---

### consultoAPI
**Descrição:** Aplicativo para consulta e comparação de CEPs com dados do IBGE, permitindo salvar e visualizar resultados.

---

### extratoPedidoZap
**Descrição:** Extração automática de pedidos estruturados (JSON) a partir de mensagens de texto não estruturadas.

---

### orgpasta
**Descrição:** Script que organiza automaticamente arquivos de uma pasta em subpastas categorizadas por tipo.
